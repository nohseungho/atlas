# ATLAS PM Direction

## 목표

ATLAS는 AI 블로그가 아니라 **수익화 콘텐츠 엔진**이다.

## 수익 흐름

검색 → 클릭 → Hook → 계속 읽음 → 신뢰 → 구매

모든 기능은 이 흐름의 어딘가를 강화해야 한다.

## 현재 우선순위

1. ATLAS 단독 수익화 달성
2. 다른 프로젝트 연동은 수익화 이후

## 기능 기준

Writer / Design / SEO 기능은 다음 네 가지 중 하나를 높일 때만 구현한다.

- 체류시간
- 클릭률
- 구매전환율
- 신뢰도

## 클로드 역할

클로드는 구현자다. Sprint 범위 밖 확장, 조사, 리팩터링은 하지 않는다.
